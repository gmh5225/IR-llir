Detailed Description

Represents a module (compilation unit) in LLIR.

A module is the base element of an LLIR project. A module corresponds to a single compilation unit.

void LLIR::Module::transform ( )

Hardware transformation.

This runs the transform layer. The transform layer is in charge of converting virtual registers to real-world hardware elements. This MUST be called before using a hardware translation layer.
