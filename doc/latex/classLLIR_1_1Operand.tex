\doxysubsection{Detailed Description}
The base for L\+L\+IR operands. 

This is the base class for all L\+L\+IR operands. This class generally should not be constructed directly; instead, use the derived versions. However, in cases where any type of L\+L\+IR operand is acceptable, this class should be used as a parameter/return type.
