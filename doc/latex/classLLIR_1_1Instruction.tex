\doxysubsection{Detailed Description}
Represents an instruction in L\+L\+IR. 

This class is used to represent a structure in L\+L\+IR. Although inheritence for more specific instructions can be done, in general using this class should suffice for most instructions. If you are working in L\+L\+IR, only extend this class if you have a specific need.
