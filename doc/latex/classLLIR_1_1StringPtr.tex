\doxysubsection{Detailed Description}
An L\+L\+IR global string pointer. 

This represents a string pointer in L\+L\+IR. A string pointer consists of the name of the pointer, and the value. There should be two copies of this class\+: one for the module, and the other whenever we need to reference the string.
